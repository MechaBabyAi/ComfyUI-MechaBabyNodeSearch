# ComfyUI MechaBaby 노드 검색

> **언어 선택**: [English](README.md) | [中文](README_zh.md) | [日本語](README_ja.md) | [한국어](README_ko.md) | [Русский](README_ru.md)

ComfyUI 워크플로우를 강화하는 노드 검색, 속성 검색 및 포트 네비게이션 기능을 제공하는 확장 프로그램. Ctrl+F 단축키 지원.

## 기능

- 🔍 **노드 검색 및 네비게이션**：노드 이름으로 빠르게 검색하고 이동
- 📝 **속성 검색**：노드 속성 이름과 속성 값(숫자 및 텍스트 포함) 검색
- 🔗 **포트 텔레포트**：노드 포트를 우클릭하여 연결된 노드로 직접 이동
- ⚡ **빠른 점프 메뉴**：노드를 선택한 후 단축키(기본값: F1)를 누르면 연결된 모든 노드를 빠르게 확인하고 이동
- 🔍 **검색창 강화**: 노드 검색 대화상자(Ctrl+F)가 이제 드래그 이동 및 고정(핀)을 지원합니다
- 📌 **창 고정**：빠른 점프 메뉴나 검색 대화상자를 지정된 위치에 고정하여 지속적으로 사용
- 👁️ **자동 표시**：창이 고정된 후 자동 표시를 활성화하면 노드를 클릭할 때 자동으로 점프 목록 표시
- ⏪ **기록 탐색**：마우스 사이드 버튼 또는 사용자 정의 가능한 키보드 키로 이동 기록 탐색
- 🔄 **자동 이동**：포트에 연결이 하나만 있을 때 자동으로 이동(선택 사항)
- ⌨️ **키보드 단축키**：`Ctrl+F`를 눌러 검색 대화상자를 빠르게 열기
- 🎯 **스마트 매칭**：노드 제목, 유형, 노드 ID, 위젯 이름, 위젯 값, 속성 이름, 속성 값 검색 지원
- ✨ **골든 플래시 하이라이트**：이동 후 대상 노드가 금색으로 깜빡여 더 쉽게 찾을 수 있습니다
- 🔄 **Set/Get 노드 지원**：`easy getNode`/`easy setNode`（comfyui-easy-use）및 `GetNode`/`SetNode`（ComfyUI-KJNodes）간 이동 지원
- 📋 **위젯 값 복사**：노드 우클릭「위젯 값 복사」선택 또는 노드 선택 후 **F4**로 복사 메뉴 열기
- ⚙️ **풍부한 설정 옵션**：자동 이동, 메뉴 차단, 탐색 키, 메뉴 크기 조정 등 포괄적인 설정

## 설치

1. 확장 프로그램 디렉토리 `ComfyUI-MechaBabyNodeSearch`를 `ComfyUI/custom_nodes/` 디렉토리에 배치
2. ComfyUI 재시작
3. 확장 프로그램이 자동으로 로드됩니다

## 사용 방법

### 노드 검색

1. **키보드 단축키**：`Ctrl+F`를 눌러 검색 대화상자 열기
2. **우클릭 메뉴**：캔버스를 우클릭하고 "🔍 노드 검색 (Ctrl+F)" 선택

검색 상자에 키워드를 입력하여 검색:
- 노드 제목
- 노드 유형
- 노드 ID
- 위젯 이름(예: "steps", "cfg", "seed" 등)
- 위젯 값(예: "20", "7.5", "prompt text" 등)
- 속성 이름
- 속성 값

### 확장 프로그램 설정

확장 프로그램 설정에 액세스하는 방법:
1. **우클릭 메뉴**：캔버스를 우클릭하고 "⚙️ NodeSearch 설정" 선택
2. 설정 메뉴에서 다음을 수행할 수 있습니다:
   - 인터페이스 언어 변경(중국어, 영어, 일본어, 한국어, 러시아어)
   - 키보드 단축키 사용자 정의(기본값: Ctrl+F)

**검색 팁**:
- 화살표 키(↑↓)를 사용하여 결과 목록 탐색
- `Enter`를 눌러 선택한 노드로 이동
- `Esc`를 눌러 검색 대화상자 닫기

### 포트 텔레포트

1. **방법 1**：노드를 우클릭하고 "🔗 연결된 노드로 텔레포트"를 선택한 다음 이동할 연결 선택
2. **방법 2**：포트를 직접 우클릭(포트에 연결이 있는 경우)하면 자동으로 이동하거나 연결 목록 표시
3. **방법 3**：노드를 선택한 후 빠른 점프 단축키(기본값: F1)를 누르면 연결된 모든 노드를 표시하는 빠른 점프 메뉴가 열립니다

### 포트 텔레포트 설정

포트 텔레포트 설정에 액세스하는 방법:
1. **우클릭 메뉴**：캔버스를 우클릭하고 "⚙️ NodeSearch 설정" → "🎯 포트 텔레포트 설정" 선택
2. 설정 메뉴에서 다음을 구성할 수 있습니다:
   - **자동 이동**：포트에 연결이 하나만 있을 때 자동 이동 활성화/비활성화
   - **메뉴 표시 완전 차단**：자동 이동 시 메뉴 표시 완전 차단 활성화/비활성화
   - **기록 탐색**：마우스 사이드 버튼 및 키보드 기록 탐색 활성화/비활성화
   - **탐색 키 사용자 정의**：뒤로/앞으로 탐색의 사용자 정의 키 설정
   - **빠른 점프 단축키**：빠른 점프 메뉴의 사용자 정의 단축키 설정(기본값: F1)
   - **위젯 복사 단축키**：「위젯 값 복사」서브메뉴에서 단축키 사용자 정의(기본값: F4)
   - **메뉴 크기**：빠른 점프 메뉴의 크기 조정(50%, 75%, 100%, 125%, 150%, 또는 캔버스 줌에 따른 자동 조정)

### 창 고정 및 자동 표시

1. **창 고정**：빠른 점프 메뉴나 검색 대화상자 헤더의 📍 버튼을 클릭하여 창을 고정 위치에 고정
   - 고정 후 노드로 이동해도 창이 열려 있음
   - 노드를 전환하거나 새로운 검색을 수행해도 창 위치가 고정된 상태로 유지
   - 📌 버튼을 클릭하여 고정 해제

2. **이동 가능한 대화상자**: 빠른 점프 메뉴와 검색 대화상자 모두 제목 표시줄을 드래그하여 이동할 수 있습니다. 고정 시 위치가 기억됩니다.

2. **자동 표시**：창이 고정된 경우 👁️ 버튼을 클릭하여 자동 표시 활성화
   - 활성화하면 노드를 클릭할 때 자동으로 점프 목록이 표시됩니다
   - 다시 클릭하면 비활성화되어 수동 단축키 모드로 돌아갑니다
   - 자동 표시 버튼은 창이 고정된 경우에만 표시됩니다

## 사용 예시

### 노드 검색
- "ksampler"를 입력하여 모든 KSampler 노드 찾기
- "512"를 입력하여 512 값을 포함하는 모든 노드(너비, 높이 등) 찾기
- "steps"를 입력하여 steps 위젯이 있는 모든 노드 찾기

### 포트 텔레포트
- KSampler 노드의 "model" 입력 포트를 우클릭하여 연결된 CheckpointLoaderSimple 노드로 이동
- CheckpointLoaderSimple 노드의 출력 포트를 우클릭하여 해당 모델을 사용하는 모든 노드로 이동

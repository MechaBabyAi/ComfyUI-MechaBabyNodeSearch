# ComfyUI MechaBaby 노드 검색

> **언어 선택**: [English](README.md) | [中文](README_zh.md) | [日本語](README_ja.md) | [한국어](README_ko.md) | [Русский](README_ru.md)

ComfyUI 워크플로우를 강화하는 노드 검색, 속성 검색 및 포트 네비게이션 기능을 제공하는 확장 프로그램. Ctrl+F 단축키 지원.

## 기능

- 🔍 **노드 검색 및 네비게이션**：노드 이름으로 빠르게 검색하고 이동
- 📝 **속성 검색**：노드 속성 이름과 속성 값(숫자 및 텍스트 포함) 검색
- 🔗 **포트 텔레포트**：노드 포트를 우클릭하여 연결된 노드로 직접 이동
- ⌨️ **키보드 단축키**：`Ctrl+F`를 눌러 검색 대화상자를 빠르게 열기
- 🎯 **스마트 매칭**：노드 제목, 유형, 위젯 이름, 위젯 값, 속성 이름, 속성 값 검색 지원

## 설치

1. 확장 프로그램 디렉토리 `ComfyUI-MechaBabyNodeSearch`를 `ComfyUI/custom_nodes/` 디렉토리에 배치
2. ComfyUI 재시작
3. 확장 프로그램이 자동으로 로드됩니다

## 사용 방법

### 노드 검색

1. **키보드 단축키**：`Ctrl+F`를 눌러 검색 대화상자 열기
2. **우클릭 메뉴**：캔버스를 우클릭하고 "🔍 노드 검색 (Ctrl+F)" 선택

검색 상자에 키워드를 입력하여 검색:
- 노드 제목
- 노드 유형
- 위젯 이름(예: "steps", "cfg", "seed" 등)
- 위젯 값(예: "20", "7.5", "prompt text" 등)
- 속성 이름
- 속성 값

**검색 팁**:
- 화살표 키(↑↓)를 사용하여 결과 목록 탐색
- `Enter`를 눌러 선택한 노드로 이동
- `Esc`를 눌러 검색 대화상자 닫기

### 포트 텔레포트

1. **방법 1**：노드를 우클릭하고 "🔗 연결된 노드로 텔레포트"를 선택한 다음 이동할 연결 선택
2. **방법 2**：포트를 직접 우클릭(포트에 연결이 있는 경우)하면 자동으로 이동하거나 연결 목록 표시

## 사용 예시

### 노드 검색
- "ksampler"를 입력하여 모든 KSampler 노드 찾기
- "512"를 입력하여 512 값을 포함하는 모든 노드(너비, 높이 등) 찾기
- "steps"를 입력하여 steps 위젯이 있는 모든 노드 찾기

### 포트 텔레포트
- KSampler 노드의 "model" 입력 포트를 우클릭하여 연결된 CheckpointLoaderSimple 노드로 이동
- CheckpointLoaderSimple 노드의 출력 포트를 우클릭하여 해당 모델을 사용하는 모든 노드로 이동

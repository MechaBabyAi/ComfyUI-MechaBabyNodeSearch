# MechaBaby 节点搜索

为您的 ComfyUI 工作流提供强大的搜索和导航工具。

## 核心功能

- **🔍 节点搜索 (Ctrl+F)**：通过标题、类型、ID 或控件值立即找到任何节点。支持普通、通配符（* ?）、正则三种模式。
- **🔗 端口传送**：在任何端口上点击右键，直接跳转到其连接的节点。
- **⚡ 快速跳转 (F1)**：一键查看选中节点的所有输入和输出连接。
- **📋 复制控件值 (F4)**：右键「复制控件值」或选中节点按 F4，可复制模型名、LoRA 名等到剪贴板。
- **📌 钉住与移动**：将搜索或跳转窗口固定在屏幕任何位置，方便连续操作。
- **✨ 金色闪烁**：跳转后目标节点会发出金色闪烁高亮，确保您能立即定位。

## 使用技巧

1. **搜索**：按 `Ctrl+F` 输入关键词，可在搜索框下方切换普通、通配符、正则模式，使用上下键切换，回车跳转。
2. **跳转**：选中节点按 `F1` 或直接在端口上点右键。
3. **复制**：选中节点按 `F4` 打开复制菜单，或右键选择「复制控件值」。
4. **设置**：在画布空白处右键 -> `⚙️ 节点搜索设置`。
